# 📊 Sales Data Analysis using Python

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a sales dataset using Python.  
The goal is to analyze sales performance, identify top products, and understand regional and category-wise trends.

---

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset Information
The dataset contains the following columns:
- Date
- Product
- Category
- Quantity
- Price
- Region

---

## 📊 Key Objectives
- Clean and preprocess sales data
- Create new features like Revenue
- Analyze sales trends over time
- Identify top-performing products
- Study category-wise and region-wise performance
- Visualize insights using graphs

---

## 🔍 Steps Performed

### 1. Data Cleaning
- Handled missing values
- Converted date column to datetime format
- Removed duplicates

### 2. Feature Engineering
- Created new column: Revenue = Quantity × Price

### 3. Exploratory Data Analysis
- Total revenue calculation
- Monthly sales trends
- Top products analysis
- Category-wise performance
- Region-wise sales distribution

---

## 📈 Key Insights
- Electronics is the highest revenue-generating category
- Laptop is the top-performing product
- North and South regions show strong sales performance
- Revenue is driven more by product price than quantity

---

## 📊 Visualizations
- Bar chart: Top products
- Pie chart: Category distribution
- Bar chart: Region-wise sales
- Line chart: Sales trends
